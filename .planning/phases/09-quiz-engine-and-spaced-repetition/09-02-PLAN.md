  <action>
    Create `src/lib/quiz/quizEngine.ts` with these functions:

    **getDueCards(mode: 'daily' | 'kaavya', kaavyaId?: number): Promise<VocabItem[]>**
    - If mode is 'daily': query `db.vocabItems.where('due').belowOrEqual(new Date().toISOString()).toArray()`
    - If mode is 'kaavya': query `db.vocabItems.where('kaavyaId').equals(kaavyaId)` then filter by `isDue(item.due)`
    - Return the filtered due items
    - For 'daily' mode, also include items with state=State.New (they have no due date set meaningfully yet -- all new cards are "due")

    **generateQuizQuestions(dueItems: VocabItem[], allVocab: VocabItem[]): QuizQuestion[]**
    - Shuffle dueItems and take up to 10
    - For each item:
      - correctAnswer = item.mwDefinitions[0] || item.apteDefinitions[0] || item.stem (fallback)
      - allMeanings = [...item.mwDefinitions, ...item.apteDefinitions] (deduplicated)
      - grammarFacts = { wordType: item.wordType, vibhakti: item.vibhakti, dhatu: item.dhatu, gana: item.gana, linga: item.linga }
      - Distractor pool: collect first MW/Apte definition from OTHER vocabItems (allVocab minus current item), use pickRandom from study/quiz.ts to pick 3
      - If distractor pool < 3 items, pad with remaining allMeanings from current word or use stem as last resort
      - options = shuffle([correctAnswer, ...distractors])
    - Return QuizQuestion[]

    **getMasteryStats(kaavyaId?: number): Promise<MasteryStats>**
    - Query vocabItems (filtered by kaavyaId if provided, otherwise all)
    - Count: total, newCount (state===State.New), learning (state===State.Learning), review (state===State.Review), mastered (state===State.Review && stability > 30), dueNow (due <= now)
    - Return MasteryStats object
