NOT as a Card object. Before calling scheduleReview(card, rating), you MUST reconstruct a Card via:
```typescript
const card = storableToCard({
  due: item.due,
  stability: item.stability,
  difficulty: item.difficulty,
  elapsed_days: item.elapsedDays,
  scheduled_days: item.scheduledDays,
  reps: item.reps,
  lapses: item.lapses,
  state: item.state,
  last_review: item.lastReview,
});
const updatedCard = scheduleReview(card, rating);
const storable = cardToStorable(updatedCard);
// Then update IndexedDB with storable fields
```
