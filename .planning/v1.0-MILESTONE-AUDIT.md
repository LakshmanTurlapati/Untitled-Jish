## Cross-Phase Integration (14/15 Connections)

14 of 15 cross-phase exports are properly wired. 1 partial break:

### Integration Gap

**QuizView ↔ /api/distractors** (STDY-02, low severity)
- `GET /api/distractors` exists and correctly queries MW dictionary for random meanings
- `generateQuiz()` accepts optional `fallbackMeanings` parameter
- `QuizView.tsx` calls `generateQuiz(vocabulary)` without fetching or passing fallback meanings
- **Impact:** Quiz shows "Need at least 4 words" for very short passages instead of using MW fallback distractors
- **Fix:** Add `fetch("/api/distractors?count=3")` in QuizView before calling `generateQuiz(vocabulary, fallbackMeanings)`

| `GET /api/distractors` | Built for quiz fallback but not called by QuizView | See integration gap above |
